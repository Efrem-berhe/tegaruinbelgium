INTRODUCTION
------------
This module is only for the purpose to display COVID-19 data on your site. 
It has been developed with the Help of www.trackcorona.live API and Amcharts 
API implementation in Drupal 8. It contains 2 type of Blocks one display 
the Countries with Confirmed, Recovered and Affected cases as per 
live api (Corona Live Tracker Block) and another Block is for the users 
who want to display data with the Help of World map on 
their website(Corona Live World Tracker Block).

------------
No requirements!
